Analyzing Fake News using Data Science Techniques
📌 Overview

This project focuses on detecting fake news using data science and machine learning techniques.
With the rapid growth of digital media, misinformation spreads quickly, making automated detection essential.

The system analyzes news articles and classifies them as Fake or Real based on textual features.

🎯 Objective

Apply data preprocessing and feature extraction techniques
